# E-commerce AI Seller Intelligence Platform

An AI-powered full-stack platform that helps e-commerce sellers analyze market trends, competitor products, customer sentiment, and business opportunities using Amazon product data and AI-generated insights.

## Features

- Amazon product scraping using Selenium
- AI-powered search query optimization
- Customer review extraction and sentiment analysis
- Competitor product analysis
- Product pricing insights
- Best-selling product variant detection
- AI-generated business recommendations
- Marketing and branding suggestions
- React-based analytics dashboard
- Flask REST API backend
- AI chatbot for business strategy recommendations


## Tech Stack

### Backend
- Python
- Flask
- Selenium
- OpenRouter API

### Frontend
- React
- JavaScript
- HTML
- CSS
- Vite

###Data Handling
- JSON

 Project Workflow

1. User enters a product idea  
2. AI optimizes the search query  
3. Selenium scrapes Amazon product data  
4. Product reviews and ratings are analyzed  
5. Sentiment analysis is performed  
6. AI generates business recommendations  
7. Dashboard displays insights and analytics  



 API Endpoints

| Endpoint | Description |
|----------|-------------|
| `/scrape` | Start product scraping |
| `/products` | Fetch scraped products |
| `/recommendation` | Get generated recommendations |
| `/ai-recommendation` | Get AI-generated insights |
| `/chat` | AI business chatbot |
